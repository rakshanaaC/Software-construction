# Aarohi Music Platform

Welcome to the Aarohi Music Platform project.

## Directory Structure
- `Aarohi_Documentation.md`: The complete software engineering documentation for your faculty submission (Business Architecture, User Stories, Diagrams, Test Cases, etc).
- `client/`: Next.js Premium Frontend application (Black & Gold aesthetic).
- `server/`: Node.js/Express Backend containing the required `LinkedList` and `Tree` data structures for playlist organization.

## How to Run

### Frontend (Next.js)
```bash
cd client
npm run dev
```
Open `http://localhost:3000` to view the premium landing page.

### Backend (Node.js)
```bash
cd server
npm run dev
```
*(Make sure you have Node.js installed)*
